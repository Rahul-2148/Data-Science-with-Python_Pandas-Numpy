ğŸ§¹ Q. Missing values ko remove kaise karein?

ğŸ› ï¸ Use: `arr[~np.isnan(arr)]`

--------------------------------------------------

ğŸ” Q. Missing values ko replace (impute) kaise karein?

ğŸ“Œ 1. Replace with 0:
```python

```

ğŸ“Œ 2. Replace with mean:
```python

```

ğŸ’¡ `np.nanmean()` nan ko ignore karke average nikalta hai.

ğŸ“Š Q. Nan-aware special functions:

NumPy me aise kuch functions hote hain jo nan ko automatically ignore kar dete hain:

- `np.nanmean(arr)` â†’ nan ignore karke average
- `np.nansum(arr)` â†’ nan ignore karke sum
- `np.nanmax(arr)` â†’ nan ignore karke max
- `np.nanmin(arr)` â†’ nan ignore karke min
